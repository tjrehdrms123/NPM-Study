# nodemon
### nodemon 설명
 > nodemon은 디렉토리의 파일 변경이 감지되면 노드 응용 프로그램을 자동으로 다시 시작시켜 줍니다.

### nodemon 설치
```js
npm init
npm i nodemon
```

### nodemon 셋팅
```js
"scripts": {
    "nodemon": "nodemon index.js"
}
```
