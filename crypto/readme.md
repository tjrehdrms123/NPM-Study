# crypto

### 암호화에 대해서

- 해싱이란?
  > 단방향 : 암호화 후 복호화해서 원래의 비밀번호를 알 수 없다
- 암호화란?
  > 양방향 : "키"를 통해 복호화해서 원래의 비밀번호를 알 수 있다.
- 솔트 (Salt)
  - 솔트란? 암호화하기 전 원문에 임의의 문자를 덧붙히는 것을 말한다

### crypto에서 지원하는 암호화

- Hashing
- HMAC
- PBKDF2
- Ciphers
- Encoders
