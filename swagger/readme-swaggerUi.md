# swagger-ui-express

### swagger-ui-express 설명

> swagger의 ui를 꾸며주는 패키지입니다

### swagger-ui-express 설치

```js
npm install swagger-ui-express
```

### swagger-ui-express 공식 문서 핵심 구문

- jsdoc 주석으로 경로를 마크업할 수 있다.
- swagger yml 구성을 동적으로 생성하여 이 모듈에 전달하여 문서를 생성할 수 있다
